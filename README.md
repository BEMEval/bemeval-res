### üìò Overview
**BEMEval-Res** is the residential benchmark of the open-source **BEMEval** framework ‚Äî a standardized dataset and evaluation suite for evaluating large language models (LLMs) performance on **building energy modeling (BEM)** tasks.

The benchmark focuses on translating *unstructured building descriptions* into *structured/machine-readable energy modeling schemas*, enabling consistent and reproducible evaluation of AI models in the BEM domain.

---

### üß© Key Features
- **Multiple Schemas** ‚Äì Includes both **industry** and **research** schemas:
  - [HPXML](https://hpxml.nrel.gov/) ‚Äì consensus residential schema for home energy modeling
  - **EPC-Schema** ‚Äì customized normative schema based on ISO/CEN 13790 energy performance methods
- **Representative Building Cases** ‚Äì curated building descriptions from:
  - HERS L100 test case
  - NIST NZERTF (single-family)
  - NREL iUnit (apartment/multifamily)
- **Evaluation Metrics**
  - Key‚ÄìValue Overlap Rate (KVOR)
